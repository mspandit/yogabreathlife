Modularity Theme Instructions

About this theme

Modularity is a multimedia theme framework for Wordpress. The theme design was developed with users who appreciate visual browsing and prefer a minimal layout, with their photos and multimedia featured front and center. But, the theme can been used to create everything from e-commerce sites, music review sites, portfolio sites, and everything in between. In the future, we will be releasing "child themes" to provide design alternative and additional applications that will work with Modularity. You'll definitely want to subscribe to GraphPaperPress.com to get all the latest developments and enhancements for Modularity.

Best Practices

If you want to make changes to how Modularity looks and feels, than create a child theme for Modularity. Never make changes to the Modularity theme files. Why? Because when a new version of Modularity is released, all of your changes will be lost when you update Modularity. Plus, making a child theme is incredibly easy. Here is how you do it.

Installing this theme

  1. Unzip the folder contained in the archive to your computer
  2. Upload the folder and its contents to the themes directory of your blog, located in "wp-content/themes" where WordPress is installed.
  3. Click on Appearance - Themes menu from the WP admin panel, then select the theme to activate.
  4. Click on View Site at the top of the page.

Initial Setup

Modularity is a 950px wide, one or two column theme, depending on options you have chosen. We need to set up your default image and thumbnail sizes to allow for easy posting and consistent design structure. Click Settings - Media and set your thumbnail size to 150px wide by 150px tall. Set your medium size images to 590px wide and 590px tall. Set your large size images to 950px wide and 950px tall. Post thumbnails are generated automatically by the functions contained in this theme. By default, the theme pulls in the latest image uploaded to the post. There is no need to use custom fields for adding post thumbnails. Simply upload your images using Wordpress' new Add Media button, write a caption and description and click save. While the theme will pull in the thumbnail automatically, it will not add the medium or large sized image to your post automatically. For that, just click the insert image button just as you normally would. If you need additional instructions on uploading photos, inserting photo galleries and using the Wordpress interface, please view this screencast.

Theme Options Page

Click Appearance - Theme Options. On this page you can modify background colors, link colors, add your contact info, welcome message and tracking code without touch a single line of code. You can also add and remove applications, or "apps" as I like to call them. You can read more about each "app" on the theme options page.

Widgets

There are a total of four widgetized areas on this theme. The sidebar is a widget called, you guessed it, Sidebar. It appears on the homepage only when the blog app is enabled. The bottom three-column area is 100% widgetized. From Left, Bottom-Left, Bottom-Middle, and Bottom-Right. You can add and delete widgets by clicking Design - Widgets, from within your Wordpress admin panel.

Video & Multimedia

Modularity has a built-in HD video player that plays both FLV and MOV (H.264) video files. You are no longer restricted to pixelated YouTube videos. To add a video, upload the FLV or MOV (H.264) file to your server using your favorite method, like FTP. Copy and paste the link to the file into a custom field called "video". Next, add a video thumbnail. You can do this the exact way that you add regular thumbnails, except, you will want to add an image large enough to fit the maximum size of the video player. Add the video thumbnail to a custom field with key value "video-thumb". You can also add other multimedia, which can include virtually any media with embed capabilities. Enter "multimedia" as the key value and paste your YouTube, Vimeo, or any other html embed code into the value field. You can read more about the video compression H264 here. You can read more about using custom fields here.

Display Content from Specific Categories

Lets say that you want to limit the Slider or Featured apps to just display content from a specific category. Open up the featured.php and slider.php files located in modularity/library/apps/ in a text editor. Locate this line of code:

$my_query = new WP_Query("showposts=1"); ?>

Add

&category_name=News

to the end of

showposts=1

You new code should look like this:

$my_query = new WP_Query("showposts=1&category_name=News"); ?>

This will pull posts from the News category. If you want to pull posts from multiple categories, you can replace

category_name=News

with

cat=1,5,9

to pull posts from categories with the category id's of 1, 5, and 9.
